## Title
Active Learning with Gaussian Processes: A Bayesian Approach to Learning Functions

## Abstract
When data are expensive to acquire, the efficiency with which a model learns from data becomes crucial. Instead of passively collecting data and fitting a model, the most informative data points can be selected dynamically so that learning efficiency is maximized. This is the central idea of active learning. Gaussian processes provide a natural framework for this problem because they produce both predictions and uncertainty estimates for unknown functions. In this seminar, Gaussian process regression (GPR) is introduced as a probabilistic model for learning unknown functions and is shown to support active learning through acquisition functions that rank candidate measurements according to their expected value. Different acquisition functions optimize different objectives, such as reducing overall uncertainty or targeting regions expected to yield optimal outcomes. A practical example from materials design (Kim et al., 2019) is discussed, comparing acquisition strategies for identifying polymers with high glass transition temperatures. The results demonstrate that active learning can significantly reduce the number of measurements required to find high-performing materials and that the choice of acquisition function strongly influences the efficiency of the search process.
